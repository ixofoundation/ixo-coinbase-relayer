# ixo-coincodex-relayer

This is a minimal project for relaying requests to coincodex and avoiding CORS issues.

## Features

- Minimal
- TypeScript
- Wrangler to develop and deploy.

## Usage

### Install

```
yarn install
```

### Develop

```
yarn dev
```

### Test

```
yarn test
```

### Deploy

```
yarn deploy
```
